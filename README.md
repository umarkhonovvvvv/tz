# Next.js 15+ Post Application

Це невеличкий застосунок на Next.js 15+, побудований з використанням сучасних інструментів.

## 🚀 Технологічний стек
- **Framework**: Next.js 15 (App Router)
- **State Management**: TanStack Query
- **Styling**: Tailwind CSS
- **Architecture**: Feature-Sliced Design (FSD)
- **Language**: TypeScript

## 🛠 Інструкція по запуску
1. Встановіть залежності: `npm install`
2. Запустіть проект: `npm run dev`
